**jwt-auth.guard.ts:**

- Create JwtAuthGuard class implementing CanActivate
- Inject Reflector in constructor
- In canActivate:
  1. Check for @Public() metadata using reflector.getAllAndOverride
  2. If public, return true immediately
  3. Otherwise, extract Bearer token from Authorization header
  4. Verify JWT using jose (same logic as existing auth.middleware.ts)
  5. Attach user to request object
  6. Return true if valid, throw UnauthorizedException if invalid
- Reuse AuthenticatedUser and AuthenticatedRequest types from auth.types.ts

**Update main.ts:**

- Import JwtAuthGuard and Reflector from @nestjs/core
- Add app.useGlobalGuards(new JwtAuthGuard(new Reflector()))
- Remove any route-specific middleware configuration

Create the remaining common infrastructure:

**http-exception.filter.ts:**

- Implement ExceptionFilter with @Catch(HttpException)
- In catch method, extract:
  - status code from exception
  - message (handle both string and object message from ValidationPipe)
  - request URL and method
  - timestamp (ISO 8601)
- Return JSON: { statusCode, message, path, method, timestamp }
- Register globally in main.ts using app.useGlobalFilters()

**query.dto.ts:**
Create reusable QueryDto class with these validated fields:

- page?: number (default 1, min 1, use @Type(() => Number) for transform)
- limit?: number (default 20, min 1, max 100)
- sort?: string (e.g., "-createdAt" for descending)
- search?: string (optional text search)
- status?: string (optional status filter)
  All fields should use @IsOptional() decorator.

**paginated-response.dto.ts:**
Create generic PaginatedResponseDto<T> class:

```typescript
export interface PaginatedMeta {
  total: number
  page: number
  limit: number
  totalPages: number
}

export class PaginatedResponseDto<T> {
  data: T[]
  meta: PaginatedMeta
}
```

Also create a helper function:

```typescript
export function paginate<T>(
  items: T[],
  query: { page?: number; limit?: number }
): PaginatedResponseDto<T>
```

This function should:

1. Apply pagination (slice array based on page/limit)
2. Calculate totalPages
3. Return { data, meta }
