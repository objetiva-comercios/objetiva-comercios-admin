1. **DTOs**:
   - `existencia-query.dto.ts`: Extends base QueryDto pattern. Fields: `depositoId` (optional number, Transform to Number), `search` (optional string), `status` (optional: 'normal' | 'bajo' | 'sin_stock'), `page`/`limit` (with Transform+defaults). Status filter is applied in service via SQL CASE.
   - `update-existencia.dto.ts`: Fields: `cantidad` (optional number, Min(0)), `stockMinimo` (optional number, Min(0)), `stockMaximo` (optional number, Min(0)). All optional with IsOptional+IsNumber+Min decorators.
   - `create-existencia.dto.ts`: Fields: `articuloCodigo` (string, IsString, IsNotEmpty), `depositoId` (number, IsNumber), `cantidad` (number, Min(0), default 0), `stockMinimo`/`stockMaximo` (number, Min(0), default 0).

2. **Service** (`existencias.service.ts`): Inject DrizzleService. Methods:
   - `findByDeposito(depositoId, query)`: Select existencias joined with articulos (get articuloNombre, articuloSku). Filter by depositoId. Apply search filter on articulos.nombre/codigo with ilike. Apply status filter via SQL CASE (sin_stock: cantidad=0, bajo: cantidad>0 AND cantidad<=stockMinimo, normal: cantidad>stockMinimo OR stockMinimo=0). Paginate. Return PaginatedResponseDto.
   - `findMatrix(query)`: Select all existencias joined with articulos, paginated by distinct articulo. Return flat rows (client transforms to matrix). Apply search filter.
   - `findByArticulo(articuloCodigo)`: Select existencias for one articulo joined with deposito nombre. For articulo detail sheet stock section.
   - `getKpiStats()`: Single query using SQL CASE + count() for totalConStock (cantidad>0), stockBajo (cantidad>0 AND cantidad<=stockMinimo AND stockMinimo>0), sinStock (cantidad=0). Return object.
   - `upsert(dto: CreateExistenciaDto)`: INSERT ON CONFLICT (articuloCodigo, depositoId) DO UPDATE. Use Drizzle `.onConflictDoUpdate()` with `target: [existencias.articuloCodigo, existencias.depositoId]`.
   - `update(articuloCodigo, depositoId, dto)`: UPDATE with `and(eq, eq)` composite key. Set updatedAt. Throw NotFoundException if no row returned.
