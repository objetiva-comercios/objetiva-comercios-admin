**product.generator.ts:**

Define CATEGORIES constant array:
['Electronics', 'Clothing', 'Home & Garden', 'Sports', 'Books', 'Toys', 'Food & Beverage', 'Health & Beauty', 'Automotive', 'Office Supplies']

Define STATUSES array with weights:

- 'active' (85%)
- 'inactive' (10%)
- 'discontinued' (5%)

Create generateProduct(id: number): Product function:

- Use faker.seed(id) at start for deterministic generation
- sku: `SKU-${faker.string.alphanumeric(8).toUpperCase()}`
- name: faker.commerce.productName()
- description: faker.commerce.productDescription()
- price: faker.number.float({ min: 10, max: 1000, fractionDigits: 2 })
- cost: price \* faker.number.float({ min: 0.4, max: 0.7 }) (rounded to 2 decimals)
- category: faker.helpers.arrayElement(CATEGORIES)
- stock: faker.number.int({ min: 0, max: 500 })
- imageUrl: faker.image.urlLoremFlickr({ category: 'product', width: 400, height: 400 })
- status: faker.helpers.weightedArrayElement([
  { value: 'active', weight: 85 },
  { value: 'inactive', weight: 10 },
  { value: 'discontinued', weight: 5 }
  ])
- createdAt: faker.date.past({ years: 2 }).toISOString()
- updatedAt: faker.date.recent({ days: 30 }).toISOString()

Create generateProducts(count: number = 500): Product[] function:

- Return Array.from({ length: count }, (\_, i) => generateProduct(i + 1))

**inventory.generator.ts:**

Create generateInventory(products: Product[]): InventoryItem[] function:

- For each product, create corresponding inventory item
- Use faker.seed(product.id + 10000) for determinism
- id: product.id (1:1 mapping)
- productId: product.id
- productName: product.name
- sku: product.sku
- quantity: product.stock (match product stock)
- minStock: faker.number.int({ min: 5, max: 20 })
- maxStock: faker.number.int({ min: 100, max: 500 })
- location: `${faker.string.alpha(1).toUpperCase()}${faker.number.int({ min: 1, max: 99 })}-${faker.number.int({ min: 1, max: 50 })}`
- lastRestocked: faker.date.recent({ days: 60 }).toISOString()
- status: derived from quantity vs minStock:
  - quantity === 0 -> 'out_of_stock'
  - quantity <= minStock -> 'low_stock'
  - else -> 'in_stock'

```typescript
export interface MockData {
  products: Product[]
  orders: Order[]
  inventory: InventoryItem[]
  sales: Sale[]
  purchases: Purchase[]
}
```

Create seedAll(): MockData function:

```typescript
export function seedAll(): MockData {
  // Generate products first (they're the foundation)
  const products = generateProducts(500)

  // Generate related data
  const orders = generateOrders(200, products)
  const inventory = generateInventory(products)
  const sales = generateSales(150, products)
  const purchases = generatePurchases(50, products)

  return { products, orders, inventory, sales, purchases }
}
```
